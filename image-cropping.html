Demo images to crop original format portrait and landscape

Crop with CSS rule: "object-fit: cover"

Images always fit centered into surrounding container. Overlapping image (vertically or horizontally) is cut off.

Pros

  • Works with CSS only (most browsers), no JavaScript needed
  • Plays nicely together with responsive image plugin

Cons

  • Doesn't work in Internet Explorer, Polyfill needed
